reverse
=======
Reverses the given message

Parameters
----------
**Message <Rquired>**
Message you wanna reverse

**Layout:**
::
    -reverse <message> 

Permissions
-----------
**Bot**
::
    Embed Links
    Send Messages

**Author**
::
    Send Messages

Cooldown
--------
10 Seconds - Per User

Aliases
-------
This command has no aliases as of now


Example Usage
-------------
::

"image_here"
